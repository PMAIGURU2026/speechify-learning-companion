# MoSCoW Prioritization

**Speechify Learning Companion - MVP (Week 4 Build)**

*David O | Pursuit AI Fellowship*

---

## Must Have (Non-Negotiable for Launch)

| Feature | Description |
|---------|-------------|
| Text upload and TTS playback | Support .txt, .pdf uploads; paste text; neural voices |
| AI-generated quiz questions | Questions every 5 minutes of listening; 4 answer options |
| Quiz response collection and immediate feedback | Store responses; show correct/incorrect with explanation |
| User authentication and session tracking | Email/password; JWT; session persistence |

---

## Should Have (Important but not launch-blockers)

| Feature | Description |
|---------|-------------|
| Retention dashboard with score trends | Total quizzes, avg score, streak, line chart |
| Multiple voice options | Voice selection dropdown |
| Variable playback speed | 0.5x, 1x, 1.5x, 2x |

---

## Could Have (Nice-to-haves if time permits)

| Feature | Description |
|---------|-------------|
| Explanation text for incorrect answers | AI-generated explanation when user selects wrong |
| Quiz difficulty levels | Easy/medium/hard tagging |
| Export quiz history to CSV | Download for external analysis |
| Dark mode UI | Theme toggle |

---

## Won't Have (Explicitly deferred to future phases)

| Category | Items |
|-----------|-------|
| Mobile | iOS/Android native apps; offline quiz syncing |
| Social | Shared playlists; commenting; leaderboards |
| Payments | Stripe/PayPal subscription processing |
| Advanced analytics | Spaced repetition; predictive performance; topic breakdown |
| Content | Pre-loaded library; curated reading lists |
| Voice | Custom voice cloning; accent selection beyond defaults |

---

## Summary

- **Must Have:** 4 items — core learning companion mode
- **Should Have:** 3 items — dashboard and playback UX
- **Could Have:** 4 items — polish if sprint allows
- **Won't Have:** 6 categories — post-MVP roadmap
